# New Typographical Rules

These rules have been applied to the application for improved readability and comfort:

1. **Minimum Font Size:** 16-18px minimum font size for body text for comfortable reading.
2. **Line Length:** 60-70 characters per line for comfortable reading (`max-w-[65ch]`).
3. **Line Height:** 1.5-1.6 minimum line height for breathing room (`leading-relaxed` or `1.6`).
4. **Type Scale:** Consistent type scale using 16, 20, 24, 32, 48, 64px font sizes.
5. **Alignment:** Left-aligned body text for 25% faster reading.
